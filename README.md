# GestionProductosFinal
++ Programador: Fabrcio quiroga tengo 19 años y este proyecto forma parte de mi aprendizaje sobre diversos conceptos de java cree una aplicacion en java utilizando javaFx con el fin de gestionar un supermercado La herramienta permite:

. Crear, modificar y eliminar productos.
. Trabajar con distintas categorías: comida,bebida,medicamentos, electrodomesticos . Ordenarlos por nombre, precio o stock.
. Filtrarlos según un precio máximo. . Guardar la información desde archivos CSV y JSON.
. Exportar listados TXT.

creado con una interfaz simple para que cualquiera pueda utilizarlo sin necesidad de contar con conocimientos informaticos, ademas de estar etilizada por un css.
